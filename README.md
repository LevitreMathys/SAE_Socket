## Utilisation
**Compilation et exécution du serveur**
```bash
gcc serveur_base_tcp.c game_pendu.c -o serveur
./serveur
```

**Compilation et exécution du client**
```bash
gcc client_base_tcp.c pendu_ascii.c -o client
./client
```
